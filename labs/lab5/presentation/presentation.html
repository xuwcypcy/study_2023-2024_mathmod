Найдите стационарное состояние системы. # Выполнение задания ## С помощью Scilab построим график модели «хищник-жертва» В Scilab мы задаём коеффициенты, соответстующие с заданием: ``` Juila a= 0.13; // коэффициент естественной смертности хищников b= 0.33;
